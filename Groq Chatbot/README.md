# Groq AI Chatbot 🤖

A simple AI chatbot made using Python and the Groq API.

This chatbot can:
- Chat continuously
- Remember previous messages
- Generate AI responses using Llama 3

---

# Features ✨

- Continuous conversation using while loop
- Chat memory support
- Fast AI responses
- Beginner-friendly code
- Uses Groq API

---

# Technologies Used 🛠️

- Python
- Groq API
- Llama 3 Model

---

# Installation ⚙️

Install required library:

pip install -r requirements.txt

---

# Setup 🔑

Create an API key from:
https://console.groq.com/keys

Then set your API key as an environment variable.

Example:

set GROQ_API_KEY=your_api_key_here

---

# Run the Project ▶️

python chatbot.py

---

# Example 💬

You: Hello  
Bot: Hi! How are you today?
