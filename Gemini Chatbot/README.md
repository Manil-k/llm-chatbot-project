# Gemini Chatbot

A simple AI chatbot built using Google's Gemini API and Python.

## Features

- Interactive terminal-based chatbot
- Uses Gemini 2.5 Flash model
- Continuous conversation loop
- Markdown formatted responses
- Secure API key handling using Google Colab userdata

---

## Technologies Used

- Python
- Google Generative AI SDK
- Google Colab
- IPython
